# CardioIA-Fase1-Batimentos-de-Dados
Objetivo

Este projeto organiza dados utilizados para aplicações de Inteligência Artificial na área da saúde cardiovascular.

Foram coletados três tipos de dados:

dados numéricos

dados textuais

dados visuais

Esses dados poderão ser utilizados em algoritmos de Machine Learning, NLP e Visão Computacional.

Dados Numéricos

Foi criado um dataset contendo informações como:

idade

sexo

pressão arterial

colesterol

frequência cardíaca

histórico de doenças cardíacas

Essas variáveis são importantes para modelos de IA que buscam prever risco cardiovascular.

Dados Textuais

Foram utilizados textos relacionados a:

doenças cardíacas

prevenção cardiovascular

saúde pública

Esses textos podem ser analisados por algoritmos de Processamento de Linguagem Natural (NLP) para identificar sintomas, fatores de risco e tópicos médicos.

Dados Visuais

Foram reunidas imagens de exames cardiológicos (ECG).

Essas imagens podem ser analisadas por algoritmos de Visão Computacional, permitindo identificar padrões e possíveis anomalias em exames cardíacos.
